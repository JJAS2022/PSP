Creación de un Servidor Web con Node.js

Node.js facilita la creación de servidores web de manera sencilla. A continuación, se explica paso a paso cómo crear un servidor básico utilizando módulos esenciales.

Importación de Módulos

En Node.js, los módulos son bloques de construcción esenciales. Para crear un servidor, necesitamos importar algunos módulos clave, como http, fs y path, así como posiblemente módulos propios que hayamos creado para funciones personalizadas.

Creación de un Servidor

La función principal para crear un servidor en Node.js es createServer(). Esta función toma como parámetro un requestListener, una función que será ejecutada cada vez que el servidor reciba una solicitud. Utilizaremos los objetos request y response para manejar la solicitud y enviar la respuesta. 


                const http = require('http');

                const requestListener = function (request, response) {
                    // Manejo de la solicitud y respuesta aquí
                };

                const server = http.createServer(requestListener);

Después de crear el servidor, debemos decirle que esté a la escucha en un puerto específico utilizando la función listen(). Esta función puede tomar un número de puerto como parámetro. 


                const puerto = 80;
                server.listen(puerto, () => {
                    console.log(`El servidor está escuchando en el puerto ${puerto}`);
                });
