Creating a Web Server with Node.js

Node.js makes it easy to create web servers. Below, we explain step by step how to create a basic server using essential modules.

Importing Modules

In Node.js, modules are essential building blocks. To create a server, we need to import some key modules, such as http, fs, and path, as well as possibly custom modules we've created for personalized functions.

Creating a Server

The main function to create a server in Node.js is createServer(). This function takes a requestListener as a parameter, a function that will be executed each time the server receives a request. We will use the request and response objects to handle the request and send the response. 


                const http = require('http');

                const requestListener = function (request, response) {
                    // Handle request and response here
                };

                const server = http.createServer(requestListener);

After creating the server, we need to tell it to listen on a specific port using the listen() function. This function can take a port number as a parameter. 


                const port = 80;
                server.listen(port, () => {
                    console.log(`The server is listening on port ${port}`);
                });
